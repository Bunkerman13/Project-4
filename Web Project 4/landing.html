Moving Words

The moving words page is the first of a three part interactive web expeience. This section is so that the user may pick a word that they may want to learn about. The words are moving back and forth and the user must click on one in order to progress to the next section.

Display Info

This is the final page of the three part interactive web experience. This section takes the word and definition and displays it along with other info. Like information about how the word came to be, some gifs about the word and even some audio of the word and definition.

Find Words

The find words page of this project is the second part of this interactive web experience. This section has the user find the definition for the word picked previously while it is hidden. When the user hovers over the word it increases in size and changes color so the user can check it. If the user clicks and it's the wrong, it turns red, if it's right, it turns green and a continue button shows.
